FileMaker plug-in API (Headers/FMWrapper/ and Libraries/)
---------------------------------------------------------
The FMWrapper headers and link libraries are part of the Claris FileMaker
External Plug-In API, Copyright (c) Claris International Inc. (see the notice at
the top of each header file). They come from the official FileMaker Plug-In SDK
(API VERSION 77) and are redistributed here under the BSD-style license printed
in each header, which permits redistribution in source and binary forms provided
the copyright notice is retained. These files are NOT covered by the ZooPlug
license above. The bundled libraries are:

    Libraries/win64/FMWrapper.lib        Windows x64 (import library)
    Libraries/macOS/FMWrapper.framework  macOS (universal: x86_64 + arm64)

FMWrapper is only a stub: it resolves at run time against the FMWrapper that ships
inside FileMaker, so the built plug-in is only useful to licensees of Claris
FileMaker. The Linux (libFMWrapper.so) and iOS libraries are NOT bundled here —
obtain them from the Claris FileMaker Plug-In SDK if you need them. (iOS is not a
target anyway: moo_shell cannot launch a process in the iOS sandbox.)
